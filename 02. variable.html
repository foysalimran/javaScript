<script> var a = "LowerCase variable"; var A = "UpperCase variable"; document.write('<div class="text-center">',a,'</div>', "<br>"); document.write('<div class="text-center">',A,'</div>', "<br>"); </script>

* Nameing

  1. We should use meaningful name.
  2. We can use (a-z) or (A-Z) letters, (0-9) numbers, _ underscore, or $ doller sign to decleare a variable.
  3. Variable name should start with letters, _ underscore, and $ doller and it can't be start with any numbers.
  4. javaScript variables are case sencetive, it's mean name and NAME is totally two different variables.
  5. We can't use any resurve keywords as variable name. example: var, let, function, else etc. can't be use as variable name. Check resurve keywords full list here

Best practice for nameing is: aLongNameHere or a_long_name_here

We can set value multiple time as we need. Example:

<script> myAge = 22; document.write('<div class="text-center">',myAge,'</div>', "<br>"); myAge = 25; document.write('<div class="text-center">',myAge,'</div>', "<br>"); </script>
